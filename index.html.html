🔥 Firebase 연결 설정

실시간 팀 공유를 위해 Firebase 프로젝트 정보를 입력해주세요.
한 번만 설정하면 이후 자동으로 연결됩니다.

Firebase 설정 방법 (무료)
1. console.firebase.google.com → 프로젝트 만들기
2. 왼쪽 메뉴 → 빌드 → Realtime Database → 데이터베이스 만들기 (테스트 모드)
3. 프로젝트 설정(⚙️) → 일반 → 내 앱 → 웹앱 추가 → apiKey, databaseURL 복사
